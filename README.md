## Issues Encountered

### GitHub Registration — CAPTCHA Failure
- **Problem:** Unable to complete GitHub sign-up due to persistent CAPTCHA failures during registration. The CAPTCHA puzzle would not resolve despite multiple attempts.
- **Troubleshooting steps taken:**
  - Tried multiple browsers (Chrome, Firefox, Incognito mode)
  - Disabled all browser extensions (ad blockers, etc.)
  - Switched between VPN on/off (Singapore server)
  - Tried on mobile browser and GitHub mobile app
  - Switched between WiFi and mobile hotspot
  - Cleared browser cache and cookies
- **Solution:** Used a different device on a different network — registration succeeded immediately. The issue was likely that my original device/network was being flagged by GitHub's bot detection system.
